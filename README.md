# Market AI - Discord Marketplace Bot

بوت Discord احترافي متكامل للسوق التجاري مع ذكاء اصطناعي — بواجهة ألواح تحكم (Panel-based) ونظام دفع ProBot Credits.

## 🚀 المميزات

### 🏪 نظام المتاجر
- إنشاء متاجر كاملة مع Categories وقنوات مخصصة
- أنواع متاجر: مجاني، VIP، مميز، موثق
- نظام عمولات متدرج حسب نوع المتجر

### 📦 نظام المنتجات والخدمات
- منتجات رقمية وفيزيائية مع مخزون ومتغيرات
- خدمات مع باقات ووقت تسليم وتعديلات
- عرض احترافي مع Embeds وأزرار تفاعلية

### 💳 نظام الدفع والمحفظة
- محفظة داخلية مع إيداع وسحب وتحويل
- دعم كريدت برو بوت مع تأكيد تلقائي عبر Webhook
- سجل معاملات كامل

### 💸 نظام العمولات
- مجاني: 10% | VIP: 5% | مميز: 3% | موثق: 1%
- حساب استلام عمولات مركزي
- تحصيل آلي للعمولات

### ⭐ نظام التقييمات
- تقييم بالنجوم + تعليق نصي
- مميزات وعيوب
- رد البائع على التقييم
- تصويت على فائدة التقييم

### 🔍 نظام البحث المتقدم
- بحث في المتاجر، المنتجات، الخدمات، البائعين
- فلاتر: الفئة، السعر، التوفر، الخصم، الترتيب
- اقتراحات ذكية بالذكاء الاصطناعي

### 🛒 Marketplace تلقائي
- قناة واحدة تعرض: أفضل المتاجر، المنتجات الرائجة، أحدث المنتجات، الأعلى تقييماً
- تحديث تلقائي كل 5 دقائق
- أزرار تصفية وفلترة تفاعلية

### 🎫 نظام التذاكر
- أنواع: دعم، بلاغ، نزاع، شراكة، توثيق، تقني، فوترة
- أولويات وتخصيص للموظفين
- نسخة محادثة وتقييم رضا

### 🏆 نظام الثقة
- مستويات: موثق ✅ | موثوق 🏆 | مميز 💎
- متطلبات واضحة لكل مستوى
- لوحة متصدرين للبائعين

### 🎁 نظام الولاء
- نقاط على كل عملية شراء وتقييم وإحالة
- مكافآت: خصومات، إعفاء عمولة، ترقية متجر، شارات
- متجر مكافآت تفاعلي

### 📊 لوحة التحكم
- إحصائيات شاملة: مبيعات، إيرادات، عمولات، مستخدمين
- تقارير إيرادات بفترات زمنية
- أعلى الأداء: متاجر، منتجات، خدمات، بائعين

### 🤖 الذكاء الاصطناعي (Groq AI - مجاني)
- محادثة عامة: برمجة، دراسة، إبداعي، ترجمة، تلخيص
- مساعدة المنتجات: وصف، سعر، إعلان، كلمات مفتاحية
- مساعدة المشتري: بحث ذكي واقتراحات مخصصة
- سجل محادثات محفوظ

### 🛡️ الحماية والأمان
- Anti-spam مع تحذيرات وحظر تلقائي
- Anti-scam كشف كلمات مشبوهة
- Rate limiting مرن
- تحقق من الملكية للعمليات الحساسة
- نظام Cooldown للألواح

## 🛠️ المتطلبات التقنية

- Node.js 18+
- MongoDB 6+
- Redis (اختياري - للكاش)
- Groq API Key (للذكاء الاصطناعي - مجاني)
- Discord Bot Token

## 📦 التثبيت والتشغيل

### محلياً (Local)

```bash
git clone <repo-url>
cd market-ai
npm install
cp .env.example .env
# تحرير .env وإضافة القيم المطلوبة
npm run deploy
npm start
```

### Docker

```bash

docker compose logs -f bot
docker compose down
```

## 🎯 الألواح الرئيسية (6 Panels)

| اللوح | الوصف | الصلاحية |
|-------|-------|----------|
| `/market` | 🏪 السوق: متاجر، منتجات، خدمات، بحث، محفظة، تقييمات، ولاء | الجميع |
| `/ai` | 🤖 الذكاء الاصطناعي: محادثة، برمجة، تصحيح، شرح، تلخيص، ترجمة | الجميع |
| `/ticket` | 🎫 التذاكر: إنشاء، سحبي، دعم، بلاغ، شراكة | الجميع |
| `/profile` | 👤 الملف الشخصي: الرصيد، الأرباح، الولاء، الثقة، المشتريات | الجميع |

## ⚙️ متغيرات البيئة

```env
# Discord
DISCORD_TOKEN=your_bot_token
CLIENT_ID=your_application_id
GUILD_ID=your_guild_id
OWNER_ID=your_discord_id

# Database
MONGODB_URI=mongodb://localhost:27017/market-ai

# Redis (اختياري)
REDIS_HOST=localhost
REDIS_PORT=6379

# Groq AI (Free)
GROQ_API_KEY=gsk_your_groq_key
GROQ_MODEL=qwen/qwen3-32b

# ProBot Credits Payment
PROBOT_ACCOUNT_ID=your_probot_account_id

# Webhook Server (للتأكيد التلقائي)
WEBHOOK_PORT=3000
WEBHOOK_SECRET=your_secret

# ProBot Auto-Confirm
AUTO_CONFIRM_ENABLED=false
AUTO_CONFIRM_POLL_INTERVAL=30000

# Server
PORT=3000
LOG_LEVEL=info
```

```
src/
├── commands/       # أوامر Slash Commands + معالجات الألواح

├── database/models/ # نماذج Mongoose
├── services/      # خدمات (AI، دفع، سحوبات، عمولات، مراقبة)

├── utils/         # أدوات مساعدة (Embeds، PanelManager، تحقق، أخطاء)
├── middleware/     # وسطاء (الأمان، التحقق)
├── config/        # التكوين
└── index.js       # نقطة الدخول
```

## 🛡️ الأمان

- جميع العمليات المالية عبر MongoDB transactions (atomic)
- نظام Nonce للأزرار الحساسة (يمنع التلاعب)
- كشف المعاملات المكررة والاحتيال
- سجل تدقيق كامل (AuditLog) لكل عملية مالية
- صلاحيات صارمة: Admin للميزات الحساسة، Owner لإعدادات النظام
- Rate limiting و Anti-spam و Anti-scam
- تحقق من صحة البيئة عند بدء التشغيل
